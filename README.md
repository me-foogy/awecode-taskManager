# TASK MANAGER
___
## Indivisual project done to practise Javascript, Typescript and DOM manipulation
___
### PROJECT DESCRIPTION
This project includes a simple task manager that uses subtle colors and styles and focuses on functionality.
Users can simply add tasks, place them into completion state and delete them if required.

Passive features (V1.0.0)
- Responsive on both mobile and web.
- Due Date automatically set to today's date when user does not specify it.
